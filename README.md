# Qodalis CLI — Vue Example

Minimal Vue 3 example using [`@qodalis/vue-cli`](https://www.npmjs.com/package/@qodalis/vue-cli).

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/qodalis-solutions/stackblitz-qodalis-cli-vue-example)

## Quick Start

```bash
npm install
npm run dev
```

## What's Included

- `@qodalis/vue-cli` — Vue 3 wrapper (Cli, CliPanel, CliConfigProvider)
- `@qodalis/cli-guid` — GUID generator plugin
- `@qodalis/cli-files` — File browser plugin
- `@qodalis/cli-users` — User management plugin
